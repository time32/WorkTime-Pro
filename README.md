# WorkTime Pro - 个人工时记录助手 ⏳

**WorkTime Pro** 是一款基于 Python (Tkinter + ttkbootstrap) 的桌面工时记录工具，专为工业自动化工程师及办公人员设计。它支持自动扣除午休时间、工时取整计算（0.5h）以及开机自启功能。

## ✨ 主要功能

- **智能计算**：自动扣除午休（12:00-13:00），工时向下取整至 0.5 小时。
- **状态管理**：支持“上班/下班/中途外出”记录，实时显示当前状态。
- **月度视图**：直观的日历界面，统计当月应出勤、实际出勤及缺勤时长。
- **本地存储**：基于 SQLite 数据库，数据安全存储在本地。
- **现代化 UI**：采用扁平化设计（ttkbootstrap），支持深色悬浮菜单。
